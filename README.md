Телефонний довідник на Laravel/

Додаток призначений для відображення робочої інформації щодо внутрішніх та міських телефонів, посад , відділів, географічної адреси, та адреси електроної пошти співробітників, і розрахований на використання в локальній мережі організації. 
На боці кліента не потребує ніяких додаткових програм чи компонентів, доступ відбувається через Веб-броузер.
Додаток підтримує авторизацію користувачів, але не примушує до її виконання.

Для отримання доступу редагування даних необхідно зареєструватися та опанувати роль Адміністратора (поле role = 'admin', що виконується напряму в БД). Після цього для означеного користувача буде доступний інтерфейс Адміністратора.
